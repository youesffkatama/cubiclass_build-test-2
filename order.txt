# SCHOLAR.AI RECREATION PROMPT FOR AI SYSTEMS

## MISSION
Create a perfect replica of the Scholar.AI application with identical theme, style, creativity, colors, power, and features. Recreate every aspect of the original application while maintaining the same innovative spirit and functionality.

## APPLICATION IDENTITY
- Name: Scholar.AI
- Purpose: AI-powered academic workspace with RAG capabilities
- Theme: Modern educational technology platform
- Style: Clean, professional, academic with engaging elements
- Creativity: Innovative approach to learning with AI integration
- Colors: Professional academic palette (likely blues, whites, grays with accent colors)
- Power: Advanced AI features, real-time processing, gamification
- Core Features: PDF Intelligence, RAG Chat, Analytics, Gamification, Classroom Tools

## CORE FUNCTIONALITY REQUIREMENTS

### AUTHENTICATION SYSTEM
- User registration with email verification
- Secure login/logout functionality
- JWT-based authentication
- Password reset capability
- Profile management
- Session management

### PDF INTELLIGENCE ENGINE
- PDF upload capability (up to 50MB files)
- Progress indicators during processing
- OCR fallback for scanned documents
- File management system
- Processing status tracking
- Error handling for corrupted files
- Preview functionality

### RAG CHAT SYSTEM
- Real-time chat interface with streaming responses
- Context-aware Q&A based on uploaded documents
- Conversation history management
- Message threading
- Typing indicators
- Response citations/references
- Ability to switch between different document contexts

### GAMIFICATION SYSTEM
- XP points for various activities:
  - Login (+0 XP)
  - Upload PDF (+50 XP)
  - Chat Message (+2 XP)
  - Complete Task (+10 XP)
- Level calculation: level = floor(sqrt(xp / 100)) + 1
- Rank tiers: Novice → Scholar → Researcher → Professor → Nobel
- Daily streak tracking
- Achievement badges
- Progress bars and visual indicators
- Leaderboard functionality

### ANALYTICS DASHBOARD
- Study performance tracking
- Time spent on different activities
- Document interaction statistics
- Learning progress visualization
- Productivity metrics
- Interactive charts and graphs
- Export functionality for reports

### CLASSROOM FEATURES
- Create/join classes functionality
- Class stream for announcements
- Assignment management
- Student/teacher role differentiation
- Collaboration tools
- Resource sharing
- Class analytics

### STUDY TOOLS
- AI-generated flashcards from documents
- Quiz generation capability
- Study plan creation
- Note-taking functionality
- Bookmarking important sections
- Highlighting tools
- Search within documents
